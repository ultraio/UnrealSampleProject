# Ultra's Sample Project for the Unreal Online Subsystem

You can use this project to test Ultra's Unreal online subsystem which allows developers to connect the Ultra client to Unreal-based games.

## Get the Online Subsystem

You can get the latest version of Ultra's Unreal Online Subsystem from the sister [Github repository which is located here](https://github.com/ultraio/UltraUnrealOnlineSubsystem).
